One, two, buckle your shoe

11:15 AM - 11:40 AM on August 16, 2015, Room 704

Kat Chuang

Audience level:
novice
Watch:
http://youtu.be/9KWmoc285rg

Description

Programming is more than just typing, and takes more than talent. Producing software takes time, dedication, and teamwork. This talk gives guiding points for those learning to program and a demonstration of setting up a Flask web app.

Abstract

You've learned from experience to double check your shoelaces before running. When it comes to programming it isn't always the rocket science, but rather learning the best practices and safeguarding from potential risks. This talk introduces guiding points for learning to program. That is to say, you can start learning to program even if you don't have a CS degree. Following the talking points is a demonstration of writing an app from start to finish. Slides and code will be available on Github.
