State of the library: matplotlib

11:15 AM - 11:40 AM on August 16, 2015, Room 702

Thomas A Caswell

Audience level:
novice
Watch:
http://youtu.be/NYashPgYXB8

Description

A short summary of recent developments of matplotlib and how to write re-usable plotting code.

Abstract

A 'state of the project address' for matplotlib, the popular plotting library in the scientific stack. This talk will provide a brief tour of the new features in the 1.4, 1.5, and 2.0 series and a summary of the features in the pipeline for 2.1. The talk will end with a brief tutorial of how to write re-usable plotting functions.
